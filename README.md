# AI Desktop Assistant

A Python-based AI Desktop Assistant capable of voice interaction, speech recognition, and browser automation.

## Features

- Voice recognition
- Text-to-speech response
- Open Google using voice commands
- Open YouTube using voice commands
- Personalized greetings
- Continuous listening mode

## Technologies Used

- Python
- SpeechRecognition
- PyAudio
- pyttsx3
- webbrowser

- "hello"
- "your name"
- "open google"
- "open youtube"
